# STM32 综合项目创意

> 基于江科大STM32课程全部学完后，可实现的创意项目

## 已掌握技能清单

| 类别 | 外设/功能 |
|------|----------|
| 基础IO | LED、蜂鸣器、按键、光敏传感器 |
| 显示 | OLED (I2C/SPI) |
| 计数 | 红外对射、旋转编码器 |
| 定时器 | 定时中断、PWM、呼吸灯、舵机、电机 |
| 测量 | 输入捕获、频率/占空比、编码器测速 |
| ADC | 单通道/多通道 |
| DMA | 数据转运、ADC+DMA |
| 串口 | 发送/接收、HEX/文本数据包 |
| I2C | 软/硬件I2C、MPU6050 |
| SPI | 软/硬件SPI、W25Q64 |
| RTC | 实时时钟 |
| 功耗 | 睡眠/停止/待机模式 |
| 看门狗 | IWDG、WWDG |
| 存储 | 读写内部FLASH |

## 推荐项目

### 🥇 手势控制机械臂
**技能组合**：旋转编码器 × 2 + 串口DMA + PWM × 3 + ADC + OLED  
**玩法**：PC端发送角度指令或编码器手动控制，记录到FLASH

### 🏠 智能家居网关
**技能组合**：串口 + DMA + FLASH + RTC + 低功耗 + 看门狗  
**玩法**：接收传感器数据，上报PC，OLED显示

### 🎯 MPU6050平衡球
**技能组合**：硬件I2C + 编码器接口 + PWM × 2 + DMA + PID  
**玩法**：MPU6050检测倾斜，控制电机保持平衡

### 🎮 蓝牙遥控小车
**技能组合**：串口(HC-05) + PWM × 2 + 编码器 + ADC + 中断 + 看门狗  
**玩法**：手机APP控制，自动避障

### 🎵 音乐律动灯
**技能组合**：ADC + DMA + FFT + PWM + 定时器  
**玩法**：音频输入，OLED显示频谱，LED随节奏变色

### 🌙 倒计时番茄钟
**技能组合**：定时器 + RTC + FLASH + OLED + 按键 + 蜂鸣器  
**玩法**：25分钟倒计时，蜂鸣器提醒

## 建议

首选 **手势控制机械臂**：难度适中、成就感强、复用所有技能、可扩展蓝牙/存储/上位机。
