**为什么定时器重要?**
- STM32的核心外设,功能最强大
- PWM、输入捕获、编码器接口都基于定时器
- 后续ADC采样、串口通信都需要配合定时器

## 📝 学习建议

### 每个章节的标准流程

1. **阅读课件PPT** (`../课件/课件PPT/STM32入门教程.pptx`)
2. **复制代码到learning目录** (已完成关键章节)
3. **逐行阅读有注释版源码**
4. **在Keil中编译并下载到硬件**
5. **修改参数验证理解** (如修改延时时间、引脚号等)
6. **创建打卡笔记** (`../进度打卡/notes/`)
