# AI Studio STM32 产品设计提示词系统

## 核心目标

基于 STM32F103C8T6 的外设能力，设计符合实际需求的产品，并生成营销图片。

---

## 功能模块（按外设划分）

### 输入模块
| 外设 | 能力 | 产品应用场景 |
|------|------|-------------|
| GPIO 输入 | 按钮、传感器电平 | 手动控制、状态检测 |
| EXTI | 外部中断 | 紧急停止、事件触发 |
| ADC | 模拟量采集 | 温度、光敏、电位器 |
| 外部中断 | 防抖、边沿触发 | 按键消抖、脉冲计数 |

### 输出模块
| 外设 | 能力 | 产品应用场景 |
|------|------|-------------|
| GPIO 输出 | LED、继电器 | 指示灯、开关控制 |
| PWM | 模拟输出控制 | 调光、调温、电机速度 |
| DAC | 模拟电压输出 | 信号生成、音频 |

### 通信模块
| 外设 | 能力 | 产品应用场景 |
|------|------|-------------|
| USART/UART | 串口通信 | 调试、蓝牙、WiFi 模块 |
| I2C | 双线同步通信 | OLED、温湿度传感器 |
| SPI | 高速同步通信 | Flash 存储、SD 卡、屏幕 |

### 定时模块
| 外设 | 能力 | 产品应用场景 |
|------|------|-------------|
| TIM 定时器 | 精确计时 | 倒计时、周期任务 |
| TIM PWM | 脉宽调制 | 呼吸灯、蜂鸣器频率 |
| SysTick | 系统滴答 | 精确延时 |
| RTC | 实时时钟 | 时钟、日历、闹钟 |

## 提示词模板

### Template 1: 功能启发式
```
你是一个产品设计师，专注于 STM32 嵌入式应用。

我的 STM32 开发板有以下外设能力：
- 输入：GPIO、EXTI、ADC（3通道）
- 输出：GPIO、PWM、DAC
- 显示：I2C OLED (128x64)
- 通信：UART、I2C、SPI
- 存储：SPI Flash
- 其他：RTC、看门狗

请根据 [功能组合]，设计 3 个有创意且实用的产品概念。

要求：
1. 每个产品说明核心功能
2. 列出所需的外设组合
3. 给出产品使用场景
4. 设计 3 个营销卖点
```

         ↓

         ↓
3. 产品设计 → 生成 3 个产品概念
         ↓

         ↓

## 产品创意库（参考）

### 单一功能型
- **温湿度计**：I2C 传感器 + OLED 显示
- **调光台灯**：旋钮(ADC) + PWM LED
- **倒计时器**：按键 + OLED + 蜂鸣器

### 交互型
- **密码锁**：矩阵按键 + OLED + 继电器
- **游戏机**：按键 + OLED + 音效(PWM)
- **智能闹钟**：RTC + OLED + 蜂鸣器 + 温度

### 通信型
- **蓝牙终端**：UART + OLED + 按键
- **气象站**：多传感器 + Flash 存储 + OLED

## 示例对话

### 用户：我想做一个智能台灯
```
推荐外设：
- 输入：旋钮(ADC采集)
- 输出：PWM LED + OLED 显示亮度
- 其他：无

产品概念：
1. 触控调光台灯 - 通过旋钮调节亮度，OLED 显示当前值
2. 定时睡眠灯 - 倒计时自动渐暗，OLED 显示剩余时间
3. 氛围灯模式 - 多种颜色呼吸效果，通过按键切换

生成图片提示词：
"极简风格智能台灯，STM32开发板+OLED屏幕+LED灯带，白色背景，产品展示图，柔和光线"
```

## 自定义指令（AI Studio）

你可以将以下指令添加到 AI Studio 的自定义指令中：

```
当用户询问 STM32 产品设计时：
1. 先理解用户的使用场景和需求
2. 匹配 STM32 的外设能力
3. 给出 2-3 个产品概念
4. 列出每个概念需要的外设
5. 提供营销卖点
6. 如果用户同意，生成产品视觉设计图

产品设计原则：
- 实用性优先，考虑成本和可制造性
- 交互简单直观，符合用户习惯
- 充分利用外设能力，不浪费
- 有明确的差异化卖点
```
