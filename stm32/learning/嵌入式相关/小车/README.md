## 🧠 算法核心逻辑

### 1. 基础开关控制 (LineFollower)

- **全白**: 冲冲冲！(Go Forward)
- **左黑**: 赶紧往左拉一点！(Turn Left Gently)
- **右黑**: 赶紧往右拉一点！(Turn Right Gently)
- **全黑**: 哎呀没路了，快停下！(Stop)

### 2. 平滑控制策略 (SmoothControl)

- 使用了更精细的 `digitalWrite` 组合，确保在转向时不是“死等”，而是通过单侧降速实现圆滑过渡。
